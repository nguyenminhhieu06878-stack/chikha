# 🛒 E-Commerce System

Hệ thống E-commerce với ElasticSearch, AI Recommendations, và Review System.

## 🚀 Quick Start

### Backend
```bash
cd backend
npm install
npm start
```
Server: http://localhost:3001

### Frontend
```bash
cd frontend
npm install
npm start
```
App: http://localhost:3000

## ✅ Features

### Core Features (Yêu cầu cốt lõi)
- ✅ ElasticSearch - Tìm kiếm thông minh
- ✅ Review System - Đánh giá với hình ảnh

## 🔧 Tech Stack

**Backend:** Node.js, Express, SQLite, ElasticSearch, Groq AI  
**Frontend:** React 18, TailwindCSS, Axios
